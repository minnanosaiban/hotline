# 数値特徴量で「類似決算」を検索する ― 丸紅の個別ショックを類似群との乖離で発見

![類似決算検索](img/10_similar_earnings_search/00_thumbnail.png){width="1280"}

XBRL を JSON 化した **構造化データ** から、本記事では **10 次元の数値特徴量ベクトル** を抽出し、**コサイン類似度** で「似た決算」を発見します。

「類似決算検索」の真価は **発見した類似群の CAR と、クエリ銘柄自身の CAR を比較すること** にあります。CAR（累積超過リターン ＝ 個別リターン − ベンチマーク）を 2026/3 期 announcement にも拡張し、**丸紅（8002）の類似 Top-15 が平均 CAR +2.43%・勝率 60% だったのに対し、丸紅自身は -9.39%** という強い乖離を発見します。これは「数字パターン上は同業並みの好決算なのに、市場は個別要因で売った」という **数値類似だけでは捉えきれない市場の個別反応** です。

## 類似決算検索の概要

ここまでの記事はすべて「**個別銘柄の決算データを評価**」するものでした。本記事は「**ある決算に最も似ている過去の決算**」を機械的に引き出す ― 検索エンジン的アプローチです。

### 10 次元の特徴量ベクトル設計

決算の構造化 JSON から、決算の **本質を最小次元で表現する** 10 個を選定：

| # | 特徴量 | スキーマ位置 | 意味 |
|---|---|---|---|
| 1 | `net_sales_yoy` | performance.change_pct.net_sales | 売上 YoY |
| 2 | `operating_yoy` | performance.change_pct.operating_income | 営業利益 YoY |
| 3 | `pretax_yoy` | performance.change_pct.profit_before_tax | 税引前 YoY |
| 4 | `net_income_yoy` | performance.change_pct.net_income | 純利益 YoY |
| 5 | `comprehensive_yoy` | performance.change_pct.comprehensive_income | 包括利益 YoY |
| 6 | `div_growth_pct` | dividend.actual_(current/prior).annual | 年間配当成長率 |
| 7 | `op_margin` | operating_income / net_sales | 営業利益率 |
| 8 | `net_margin` | net_income / net_sales | 純利益率 |
| 9 | `segment_count` | len(segments.current) | 事業多角化度 |
| 10 | `max_seg_share` | max(seg.net_sales) / sum | 主力依存度 |

`change_pct` 系の 5 次元と `dividend` の 1 次元が **モメンタム** を、`margin` 2 次元 + `segment` 2 次元が **収益構造とポートフォリオ** を表現します。

### パイプライン全体図

決算 JSON から 10 次元の特徴量を抽出し、z-score 正規化したうえで cosine 類似度を全ペアで計算して、最も似た決算 Top-K を返す ― という流れです。

<p class="fig-meta"><i class="fa-solid fa-expand"></i> クリックで拡大 ・ 2026.05.31作成</p>

![パイプライン全体](img/10_similar_earnings_search/01_pipeline.png){width="1200"}

**embedding API 不要・ローカル計算で完結**。1,000 銘柄の Top-15 計算は **0.1 秒**、コスト 0 円。

### 集計対象とイベント数

決算短信 JSON 327 件（segment 込み）のうち **有効特徴量 ≥7 個が揃う 232 銘柄** を対象に、2026/3 期 announcement の TDnet ログ 3,549 件から短期 CAR（[-1,+1] / [-1,+5]、[-1,+20] は未完了）を計算します。

### 本記事の実装スコープ

10 次元特徴量の z-score 正規化と cosine 類似度、丸紅・双日・ＥＮＥＯＳ を query にした類似 Top-15、2026/3 期 announcement の ad-hoc CAR、「類似群 vs 自身」の乖離による個別ショック発見、PCA 2D 投影、数値特徴量 vs LLM embedding の使い分け ― までを扱います（実 embedding API・ベクトル DB は使わず、ローカル計算で完結）。

### 丸紅 2026/3 期通期の類似 Top-15

| 5 | 6448 | ブラザー工業 | 0.896 | +5.3% | +23.5% | 0.0% | +12.51% |

**読み解き**：

- 類似 Top-15 には **総合商社 5 社**（8001 伊藤忠・8053 住友・8015 豊田通商・8031 三井・2768 双日）が並ぶ ― **業種コードを使っていないのに業種が自然にクラスタリング** されている
- **数値特徴量だけで「商社業界の決算プロファイル」を再現** できることを示しています
- 異業種では 6503 三菱電機・6448 ブラザー工業・7224 新明和工業 など、決算プロファイルが商社と似た総合電機・産業機械が並ぶ
- Top-15 の平均 CAR [-1,+5] = **+2.43%、勝率 60%** ― 全体 8,049 イベントの平均（+0.47%、勝率 48.7%）を大きく上回る **健全な決算プロファイル群**

### 丸紅自身の CAR = -9.39% ― 類似群との強い乖離

しかし丸紅自身の 2026/3 期 announcement（2026-05-01 11:00 場中発表、t=0=5/1）の CAR を計算すると：

| 期間 | 丸紅自身 | 類似 Top-15 平均 | 乖離 |
|---|---|---|---|
| CAR [-1, +1] | **-11.78%** | +2.42% | **-14.20pp** |
| CAR [-1, +5] | **-9.39%** | +2.43% | **-11.82pp** |

**Top-15 の中央値が +2.6%、勝率 60% という「健全な決算プロファイル群」に位置するにも関わらず、丸紅自身は -9.4% で急落**。これは **数字パターンでは捕捉できない個別要因** で売られたことを示しています。

個別要因の候補は、ガイダンス（2027/3 期予想）の市場予想割れ、大型 M&A・減損、あるいはセグメント分析で見た「次世代事業 +127% × 金融・リース・不動産 −54.7%」の二極化を説明会で再評価した、など。数値だけ見れば「Top-15 と同じプロファイル」なので、**こうした個別ショックを事前に予言するのは構造的に難しい** ― これが本記事の最重要発見です。

逆に言えば、**類似群は高 CAR なのに自身は低 CAR → 個別ショック発生中 → 説明会・追加 IR を最優先** という使い方ができます。

### 双日も同じ構図（弱いネガショック）

双日（2768）も類似 Top-15 平均 CAR[−1,+5] = **+2.54%・勝率 60%** に対し、自身は **−4.25%**（−6.79pp の乖離）。丸紅ほど劇的ではないが、売上は二桁成長でも利益は後退、というネガティブ要素が市場で重視された形です。

### ＥＮＥＯＳ 2026/3 期 ― 初動はポジ、5 日で反転

ＥＮＥＯＳ（5020、営業利益 +339.8%＝継続事業ベースの急回復プロファイル）を query にすると、類似 Top-15 には京セラ・松竹・大豊工業・オキサイドなど「前期低調 → 当期営利急回復」型が並びます。

| 期間 | ＥＮＥＯＳ自身 | 類似 Top-15 平均 | 乖離 |
|---|---|---|---|
| CAR [−1, +1] | **+1.36%** | −1.98% | **+3.34pp（ポジ）** |
| CAR [−1, +5] | **−4.68%** | −1.56% | **−3.12pp（反転）** |

初動 [−1,+1] は類似群を上回る「ポジショック」でしたが、5 日で逆転しました。在庫影響を除けば「実質 4,400 億維持」を上回る着地が初動で好感され、その後に高 OPM セグメントの正常化懸念が後追いした、と読めます。短期サプライズの裏で構造変化が同時進行する複層銘柄で、構造面の詳細は連載 1-3／2-6／2-8 に譲ります。

クエリ銘柄自身の CAR（赤）を、類似 Top-15 の CAR 分布（緑）に重ねると、両者の乖離 ＝ 個別ショックの有無が一目で分かります。

![CAR 分布比較](img/10_similar_earnings_search/04_car_distribution.png){width="1200"}

- **緑色破線**: Top-15 平均 [-1,+5]

このグラフが本記事の **「予測ツールとしての限界と発見ツールとしての価値」** を最も端的に表現しています。

### 特徴量空間の 2D 投影（PCA）― 類似は空間上で近接する

10 次元特徴量を PCA で 2 次元に投影：

<p class="fig-meta"><i class="fa-solid fa-expand"></i> クリックで拡大 ・ 2026.05.31作成</p>

![PCA 投影](img/10_similar_earnings_search/02_feature_space_pca.png){width="1200"}

- **★ 丸紅・双日 が中央近くにクラスタリング**
- 丸紅 Top-5（青丸）と双日 Top-5（橙四角）が **両クエリ銘柄の周辺に集中**
- PCA で寄与率 PC1+PC2 ≈ 50% 程度しかなくても、**Top-5 の近接性は視覚的に明らか**

数学的には cosine 類似度（10 次元）の Top が、2 次元 PCA でも近接する ― 高次元での近さが低次元投影でも保たれることが確認できます。

### 数値特徴量 vs LLM embedding ― 使い分け

数値特徴量と LLM embedding は、コスト・速度・解釈性・捕捉できる対象が異なり、得意分野が分かれます。

<p class="fig-meta"><i class="fa-solid fa-expand"></i> クリックで拡大 ・ 2026.05.31作成</p>

![特徴量比較](img/10_similar_earnings_search/05_numeric_vs_embedding.png){width="1200"}

| 項目 | 数値特徴量（本記事） | LLM embedding（発展） |
|---|---|---|
| 次元数 | 10 | 1,536〜3,072 |
| API コスト | 0 円 | 1 銘柄 ¥0.02〜0.10（OpenAI text-embedding-3） |
| 計算速度 | 1,000 銘柄 0.1 秒 | 同 数秒 |
| 解釈性 | **高**（次元名で説明可） | 低（次元の意味不明） |
| 捕捉できる対象 | 数字パターンのみ | **質的トピック・経営課題も** |

**推奨アプローチ**：両者をハイブリッドで使う。

```
1 次フィルタ: 数値特徴量で高速に Top-50 を絞り込む（コスト 0）
↓
2 次精密化: LLM 要約・embedding で Top-10 に絞る（コスト数円）
```

LLM 要約・embedding と組み合わせれば、**「数字は似ているが経営の物語が違う」銘柄を 2 次フィルタで排除** できます。次回はこの類似検索を未来予測へ発展させます。

## まとめ

- 本記事は **「最も似た決算 Top-K」を発見する検索エンジン的アプローチ**。10 次元の数値特徴量 + cosine 類似度（embedding API 不要・コスト 0 円・1,000 銘柄 0.1 秒）
- 丸紅 query の **類似 Top-15 に総合商社 5 社が自動クラスタリング**（業種コード不使用）。平均 CAR[-1,+5] = **+2.43%・勝率 60%** の健全プロファイル群
- しかし **丸紅自身は -9.39%（-11.82pp の乖離）** ― 数字では捕捉できない **個別ショック**。**説明会・追加 IR の重要性を逆説的に証明**
- 数値特徴量 vs LLM embedding は **ハイブリッド**（数値で高速 Top-50 → embedding で精密 Top-10）。PCA 2D 投影でも Top-5 は近接
- **ＥＮＥＯＳ**：初動 [-1,+1]=**+1.36%（ポジ）** が 5 日で **−4.68%** に反転した複層銘柄（構造面の詳細は 1-3／2-6／2-8）

次回は **過去類似決算の値動き分布から未来予測を構築** します。本記事の Top-15 の CAR を「予測分布」として扱い、「類似群と自身の乖離」の検出も組み込んで、**「典型パターン or 個別ショック」を事前判定** する枠組みに発展させます。
