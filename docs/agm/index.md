# 株主総会質疑応答

YouTube公式チャンネル（ENEOS TV）で総会の様子が公開されています。しかし、そこでの質問要旨は、実際の株主が質問した趣旨とは異なっています。


<div markdown>
<p class="center larger width-40 issue-lead sec-title">
