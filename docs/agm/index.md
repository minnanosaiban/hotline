# 株主総会

私はＥＮＥＯＳの株主です。２０２６年６月の第１６回定時株主総会に出席しました。総会の様子は、会社の公式チャンネルで公開されています。

<p class="t-toc-head">公式の要約と、実際の質問</p>

<p class="width-40 margin02">
公式チャンネルの動画には、質疑応答の内容が「質問要約」としてパネル（文字テロップ）で示されています。<b>このパネルと、出席時に記録した実際の質問を照らし合わせると、14問中10問で、質問の性質そのものがすり替えられていました。</b>
</p>
<p class="width-40 margin02">
株主が名指しで詰めた「事実を答えよ」という問いが、パネルでは「対応について」「方針について」といった、会社が答えやすい一般的な問いに置き換えられています。とりわけ、本サイトが問題提起している内部通報の件（下記1）は、パネルの言い回しが、会社が実際に返した回答の言い回しとほぼ重なっています。
</p>

<p class="larger width-40 margin02">
<i class="fa-solid fa-1"></i>　「事実で答えよ」への要約
</p>
<p class="width-40">
実際の質問（40番）は「いつ、どの法律で、どのような改正があって、支払う必要がない消費税を払わなくてよくなったのでしょうか。事実でお答えください」「『個別のことにお答えしかねる』や『適切と認識している』といったような曖昧な回答では済みません」という、名指しの追及でした。パネルは「内部通報制度に関する、個別案件への対応について伺いたい。」（43:00）となっており、会社が実際に返した「個別の事案に関する回答は差し控えます……これまでも、適切な対応がなされていると認識してございます」という回答と、ほぼ同じ言葉が使われています。同じ問いは、エネオスの従業員本人によっても壇上で繰り返されましたが（103番）、そのパネルも「個別案件を踏まえ、内部通報制度をどのように運用しているか伺いたい。」（46:56）という一般的な表現にとどまり、従業員本人が同じ疑問を社内で持っているという事実には触れていません。
</p>

<p class="larger width-40 margin02">
<i class="fa-solid fa-2"></i>　株主提案の方向が逆になった要約
</p>
<p class="width-40">
配当についての実際の質問は、配当性向という指標そのものからの転換（より安定した株主資本配当率への切替）を求めるものでした。パネルは「配当性向の維持を前提に、増配できないか伺いたい。」（53:49）となっており、提案の方向が逆になっています。
</p>

<p class="larger width-40 margin02">
<i class="fa-solid fa-3"></i>　批判の的が入れ替わった要約
</p>
<p class="width-40">
独占禁止法違反（カルテル）に関する実際の質問は、総会冒頭で謝罪がなかったことへの批判、および過去のセクハラ辞任の際にも謝罪がなかったことへの批判が中心でした。パネルは「ENEOSウイングにおけるカルテル事案について伺いたい。」（1:04:30）という、事案の説明を求める質問に置き換わっています。
</p>

<p class="width-40 margin02">
全14問の対照は、以下で確認できます（要約の文言は公式チャンネルの表示をそのまま引用しています）。
</p>
